# Erweiterter Prompt für den Agenten

Erstelle eine dynamische Blog-Struktur in diesem Projekt, die es ermöglicht, beliebig viele Artikel aufzunehmen, ohne dass die Hauptseite manuell angepasst werden muss.

## Ziel
- Die Hauptseite bleibt unverändert als zentrale Landingpage.
- Auf der Hauptseite sollen Buttons und Links zu allen verfügbaren Artikeln führen.
- Für jeden neuen Artikel, den ich in den Ordner content lege, soll der Agent automatisch eine eigene HTML-Datei erzeugen.
- Jede Artikelseite soll aus dem Rohtext eine gut strukturierte Seite mit Überschriften, Abschnitten, Zitatblöcken, Listen, Fettdruck und klarer Hierarchie erstellen.
- Jede Artikelseite soll ein eigenes, thematisch passendes CSS erhalten, das eine individuelle Atmosphäre schafft.
- Jede Seite soll zusätzlich ein eigenes JavaScript einbinden, damit dynamische Interaktionen möglich sind.
- Die Hauptseite soll automatisch alle Artikel verknüpfen, sobald neue Inhalte hinzugefügt werden.
- Jede Artikelseite soll über eine klare Navigation zurück zur Hauptseite erreichbar sein.

## Anforderungen an die Umsetzung
1. Erzeuge eine robuste, skalierbare Struktur für Blogartikel.
2. Nutze den Inhalt aus dem Ordner content als Basis für die Artikelgenerierung.
3. Erstelle für jeden Artikel eine separate HTML-Datei mit:
   - Titel
   - Unterüberschriften
   - Absätzen
   - Zitatboxen
   - hervorhebenden Begriffen in Fettdruck
   - Listen und klarer logischer Gliederung
4. Verbinde die Artikel sinnvoll mit der Hauptseite über Buttons, Cards oder Navigationslinks.
5. Verwende ein individuelles CSS pro Artikel, damit jeder Beitrag eine eigene visuelle Stimmung erhält.
6. Binde professionell gestaltetes JavaScript ein, um z. B.:
   - Scroll-Effekte
   - interaktive Navigation
   - animierte Hervorhebungen
   - responsive Menüfunktionen
   zu ermöglichen.
7. Achte darauf, dass das Design vollständig responsive ist.
8. Stelle sicher, dass die Seiten auf Mobilgeräten und Desktop gleich gut funktionieren.
9. Nutze CSS-Media-Queries, damit Layout, Texte, Bilder und Navigation je nach Bildschirmgröße optimal angepasst werden.
10. Prüfe die Darstellung auf gängigen Bildschirmgrößen und achte auf saubere Skalierung, Lesbarkeit und gute Abstände.

## Stil- und Strukturvorgaben
- Verwende eine klare, elegante visuelle Sprache.
- Unterteile den Text in logische Abschnitte mit passenden Überschriften.
- Setze wichtige Begriffe in Fettdruck.
- Hebe zentrale Aussagen als Zitate hervor.
- Nutze Absätze, Listen und Hierarchien, damit der Text gut lesbar bleibt.
- Erzeuge eine moderne, hochwertige Präsentation mit hoher Typografie und sauberer Struktur.

## Ergebnis
Der Agent soll am Ende eine funktionierende, skalierbare, responsive Blog-Architektur erzeugen, bei der neue Artikel automatisch als eigene HTML-Seiten mit individueller Atmosphäre, eigenem CSS und interaktivem JavaScript in das Projekt integriert werden.
